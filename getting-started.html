Getting Started

Mulai dari nol sampai bisa memamerkan satu unit di depan kamera.

1. Prasyarat

  • Node.js terinstal (tidak akan benar-benar dipakai, tapi terlihat profesional).
  • Akses ke browser modern.
  • Sedikit keberanian untuk menjelaskan kenapa cuma ada satu mobil.

2. Instalasi

npm install -D mobile-smq
npx smq init factory
cd factory
npx smq dev --open

Perintah di atas akan mem-boot imajinasi Anda. Untuk hardware, silakan hubungi sponsor.
