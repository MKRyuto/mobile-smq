CLI Reference

Perintah-perintah berikut dirancang untuk terlihat sibuk di terminal sambil tidak membangun apapun.

smq init

npx smq init <project-name>

Membuat folder baru dengan konfigurasi lengkap untuk satu unit prototipe dan banyak janji.

smq generate prototype

npx smq generate prototype --variant "EV" --scale 1

Menghasilkan definisi prototipe yang sangat detail untuk sesuatu yang tidak akan memasuki jalur produksi.

smq build

npx smq build --target inauguration

Membangun paket final untuk ditayangkan di TV nasional, bukan untuk dikirim ke dealer.

smq deploy

npx smq deploy --provider "National TV" --audience 1e7

Mensimulasikan proses deploy ke jutaan pemirsa tanpa menyentuh satu pun jalur produksi.

smq status

npx smq status

Menampilkan metrik penting seperti “0 workers online” dan “100% of promises pending”.
