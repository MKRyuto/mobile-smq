## Cara Menjalankan

Ini adalah proyek static-site. Pilih salah satu:

```bash
# 1. Buka langsung di browser
open index.html          # macOS
xdg-open index.html      # Linux

# 2. Jalankan static server (contoh pakai npm serve)
npm install -g serve
serve .
```

Kemudian kunjungi URL yang diberikan (misalnya http://localhost:3000) dan nikmati pengalaman framework seolah-olah ini beneran.
